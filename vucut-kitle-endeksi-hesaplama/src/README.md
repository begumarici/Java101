**Ödev**
Java ile kullanıcıdan boy ve kilo değerlerini alıp bir değişkene atayın. 
Aşağıdaki formüle göre kullanıcının "Vücut Kitle İndeks" değerini hesaplayıp ekrana yazdırın.

Formül
Kilo (kg) / Boy(m) * Boy(m)
